## 🚀 Como Usar

1. **Reinicie o app** (se estiver rodando):
   ```bash
   npm start
   ```

2. **Configure o wizard**:
   - Escolha **OpenRouter**
   - Cole sua **API Key**
   - Escolha um dos **novos modelos**!

3. **Aproveite**:
   - Teste os modelos gratuitos primeiro
   - Experimente os pagos se precisar de mais qualidade
