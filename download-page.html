📋 Requisitos do Sistema

  • Windows 10 ou Windows 11 (64-bit)
  • 4 GB de RAM (8 GB recomendado)
  • 500 MB de espaço livre em disco
  • Conexão com a internet
  • Chave API de um provedor de IA (Google, Claude, etc.)

🚀 Como Instalar

  1. Clique no botão "Instalador Completo" acima
  2. Aguarde o download terminar (~250 MB)
  3. Execute o arquivo Ultron-Setup-1.0.0.exe
  4. Siga as instruções do assistente de instalação
  5. Pronto! ULTRON está instalado e pronto para usar

🔑 Primeira Configuração

  1. Abra o ULTRON (atalho no Desktop ou Menu Iniciar)
  2. Escolha seu idioma (Português ou Inglês)
  3. Selecione seu provedor de IA preferido
  4. Insira sua chave API do provedor
  5. Configure o gateway (porta padrão: 18789)
  6. Comece a conversar com seu assistente!
