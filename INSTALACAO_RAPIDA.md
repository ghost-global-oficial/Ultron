# ⚡ Instalação Rápida do ULTRON

## Para Usuários Finais (Não-Técnicos)

### 📥 Download

**Opção 1: Instalador Completo (Recomendado)**
- Baixe: [Ultron-Setup-1.0.0.exe](https://github.com/ghost-global-oficial/Ultron/releases/latest)
- Tamanho: ~250 MB
- Instala no seu computador
- Cria atalhos automáticos

**Opção 2: Versão Portátil**
- Baixe: [Ultron-Portable-1.0.0.exe](https://github.com/ghost-global-oficial/Ultron/releases/latest)
- Tamanho: ~250 MB
- Não precisa instalar
- Executa direto de qualquer pasta

### 🚀 Instalação (3 Passos)

#### Passo 1: Baixar
1. Clique no link acima
2. Aguarde o download terminar

#### Passo 2: Instalar
1. Dê duplo clique no arquivo baixado
2. Se aparecer aviso do Windows Defender:
   - Clique em "Mais informações"
   - Clique em "Executar assim mesmo"
3. Siga o assistente de instalação:
   - Escolha a pasta de instalação (ou deixe padrão)
   - Marque "Criar atalho no Desktop"
   - Clique em "Instalar"

#### Passo 3: Configurar
1. Abra o ULTRON (atalho no Desktop)
2. Escolha o idioma (Português ou Inglês)
3. Selecione o provedor de IA:
   - **Google Gemini** (recomendado para iniciantes)
   - Claude (Anthropic)
   - OpenRouter
   - Grok (xAI)
   - OpenAI
4. Insira sua chave API:
   - [Como obter chave do Google](https://makersuite.google.com/app/apikey)
   - [Como obter chave do Claude](https://console.anthropic.com/)
   - [Como obter chave do OpenRouter](https://openrouter.ai/keys)
5. Clique em "Iniciar"

### ✅ Pronto!

Agora você pode conversar com seu assistente de IA!

## 🔑 Como Obter Chaves API

### Google Gemini (Grátis)
1. Acesse: https://makersuite.google.com/app/apikey
2. Faça login com sua conta Google
3. Clique em "Create API Key"
4. Copie a chave gerada

### Claude (Anthropic)
1. Acesse: https://console.anthropic.com/
2. Crie uma conta
3. Vá em "API Keys"
4. Clique em "Create Key"
5. Copie a chave gerada

### OpenRouter
1. Acesse: https://openrouter.ai/
2. Faça login
3. Vá em "Keys"
4. Clique em "Create Key"
5. Copie a chave gerada

## ❓ Perguntas Frequentes

### O ULTRON é grátis?
Sim, o ULTRON é 100% grátis e open source. Você só precisa pagar pela API do provedor de IA que escolher (alguns têm planos gratuitos).

### Preciso de internet?
Sim, o ULTRON precisa de internet para se comunicar com os provedores de IA.

### Meus dados são seguros?
Sim! O ULTRON roda 100% no seu computador. Suas conversas e configurações ficam apenas na sua máquina.

### Posso usar sem chave API?
Não, você precisa de uma chave API de pelo menos um provedor de IA para usar o ULTRON.

### Qual provedor é melhor?
- **Google Gemini**: Bom para iniciantes, tem plano gratuito
- **Claude**: Melhor qualidade, mas pago
- **OpenRouter**: Acesso a vários modelos, flexível

### Como desinstalar?
1. Vá em Configurações do Windows
2. Apps → Apps e recursos
3. Procure "Ultron"
4. Clique em "Desinstalar"

### O instalador é seguro?
Sim! O código é open source e pode ser verificado no GitHub. O Windows Defender pode dar aviso porque o instalador não tem assinatura digital (isso custa dinheiro), mas é seguro.

## 🐛 Problemas Comuns

### "Windows protegeu seu PC"
1. Clique em "Mais informações"
2. Clique em "Executar assim mesmo"
3. Isso acontece porque o instalador não tem assinatura digital

### "Erro ao iniciar gateway"
1. Verifique se a porta 18789 está livre
2. Tente fechar outros programas
3. Reinicie o ULTRON

### "Chave API inválida"
1. Verifique se copiou a chave completa
2. Verifique se a chave não expirou
3. Tente gerar uma nova chave

### "Não consigo instalar"
1. Execute como Administrador (clique direito → Executar como administrador)
2. Desabilite temporariamente o antivírus
3. Verifique se tem espaço em disco (precisa de 500 MB)
