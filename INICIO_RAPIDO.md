# 🚀 Início Rápido - Ultron

## ✅ Correções Aplicadas

Todas as correções foram aplicadas! O Ultron está pronto para uso.

## 📋 Passo a Passo

### 1. Iniciar o App

```bash
npm start
```

### 2. Configurar Gateway

O wizard vai abrir automaticamente. Siga os passos:

1. **Bem-vindo** → Clicar "Iniciar Configuração"
2. **Rede** → Escolher "Loopback (127.0.0.1)" (recomendado)
3. **Provedor** → Escolher "OpenRouter"
4. **API Key** → Colar sua chave do OpenRouter
5. **Modelo** → Escolher qualquer modelo da lista
6. **Teste** → Aguardar teste de API (ou pular)
7. **Finalizar** → Aguardar gateway iniciar

### 3. Copiar Token

Quando o gateway iniciar, um card vai aparecer com o token. **Copie o token!**

### 4. Abrir Chat

Clique em "Abrir Chat" e comece a conversar com a IA!

## 🔧 Solução de Problemas

### Gateway não inicia?
- Verifique se a porta 18789 está livre
- Tente fechar e abrir o app novamente

### IA não responde?
- Verifique se o token foi copiado corretamente
- Recarregue a página do chat

### Modelo não reconhecido?
- **Isso não deve mais acontecer!** Todos os modelos agora têm o prefixo correto.
- Se acontecer, reporte o bug.
